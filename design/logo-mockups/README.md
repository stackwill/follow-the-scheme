# Follow the Scheme Logo Mockups

Three monochrome-first logo directions for quick comparison.

- `fts-ligature.svg`: compact `f/t/s` ligature with a wordmark. Best if the brand should feel typographic and app-like.
- `scheme-grid.svg`: folded scheme/grid monogram. Best if the mark should connect to papers, structure, and exam routes.
- `mark-route.svg`: route/check monogram. Best if the brand should imply progress through a mark scheme.

Open `contact-sheet.html` in a browser to compare the concepts at logo, nav, and favicon-style sizes.
