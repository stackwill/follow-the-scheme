# reallycool.lol Logo Mockups

Three monochrome-first SVG logo directions for quick comparison.

- `loop-wordmark.svg`: wordmark-led concept with the double-o turned into a linked loop. Best if the brand should feel typographic, compact, and memorable at nav-bar size.
- `rcl-monogram.svg`: compact `rcl` monogram with a wordmark. Best if the brand needs a standalone app icon or favicon that is still tied to the full name.
- `dot-lol.svg`: `.lol`-forward mark. Best if the domain ending should become the distinctive brand asset.

Open `contact-sheet.html` in a browser to compare the concepts at full, small, and favicon-style sizes.
